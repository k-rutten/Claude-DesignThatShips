# Sources

Gedeelde context voor dit project. De Context Agent verwerkt alles wat hier binnenkomt.

## Folders

### meetings/
Drop hier meeting transcripties (tekst of markdown). De Context Agent leest ze,
bespreekt de inzichten met je, en verwerkt de conclusies in de relevante bestanden
(constraints.md, briefing.md, patterns.md).

Naamconventie: `YYYY-MM-DD-onderwerp.md`

### screenshots/
Drop hier referentiescreenshots voor visuele inspiratie. De Context Agent beschrijft
wat hij ziet en meldt het aan de UX Agent. Dit is de primaire visuele inspiratiebron —
jouw curatie is sterker dan welke MCP search ook.

## Losse bestanden

Zet hier ook:
- Links naar SharePoint documenten
- Figma exports
- Azure DevOps wiki pagina's
- Stakeholder feedback
- Alles wat context geeft voor het project

De Context Agent pikt alles op dat hier nieuw binnenkomt.
