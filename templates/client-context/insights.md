# [Client Name] — Inzichten

> Jouw eigen input. Schrijf hier alles wat je weet, denkt, of vermoedt over dit project.
> De Context Agent leest dit bij elke sessie en bespreekt nieuwe toevoegingen met je.

## Format

Voeg inzichten toe als bullets met een datum:

- [2025-01-15] Stakeholder X wil eigenlijk Y, niet wat er in de backlog staat
- [2025-01-15] Het huidige systeem kan geen real-time data — alles is batch
- [2025-01-16] Na meeting: scope is kleiner dan gedacht, alleen admin-rol relevant

## Status

Elk inzicht krijgt na bespreking met de Context Agent een status:
- `[confirmed]` — besproken en bevestigd, verwerkt in constraints/briefing
- `[signal]` — nog niet besproken, wordt meegenomen als context
- `[superseded by: ...]` — vervangen door een nieuwer inzicht
