# Design That Ships

Compressed design process for B2B prototyping. See [SKILL.md](SKILL.md) for the full pipeline.

## Structure

- `SKILL.md` — Pipeline orchestration
- `agents/` — Agent prompts (Context, Brief, UX, Tech)
- `references/` — Design checklist, visual composition, European B2B context
- `templates/` — Client context repo templates

## Agents

| Agent | Role |
|---|---|
| Context Agent | Session bootstrap, context refinement, knowledge capture |
| Brief Agent | Feature intake, briefing management, quality ownership |
| UX Agent | Interaction diagnosis, MCP-driven psychology |
| Tech Agent | Workarounds, prototype building, design tokens |
